# 🚀 2-DOF Rocket Flight Dynamics & Trajectory Simulation (MATLAB)

This repository contains a 2-Dimensional, 5-State flight dynamics simulation of a rocket. The project models the ascent phase, gravity turn maneuver, atmospheric drag using an exponential air density model, and continuous mass depletion.

Developed in **MATLAB R2023b**, this simulation provides a fundamental engineering framework for understanding launch vehicle trajectories.

## 🛠️ Features & Engineering Models

* **Atmospheric Drag Model:** Incorporates an exponential atmospheric density model where air density scales down dynamically with altitude.
* **Gravity Turn Maneuver:** Simulates a realistic pitchover program (vertical climb for the first 3 seconds, followed by a gradual gravity turn).
* **Mass Attenuation:** Accounts for continuous fuel consumption affecting the vehicle's moment of inertia and acceleration dynamically.
* **Numerical Solver:** Utilizes MATLAB's variable-step Runge-Kutta 4th/5th-order ODE solver (`ode45`) for high-fidelity state estimation.

---

## 📂 Repository Structure

* `main_firstclass.m`: The core script that initializes parameters, executes the ODE solver, and plots the flight telemetry.
* `rocket_dynamics.m`: The state-space function containing the equations of motion (Forces, Aerodynamics, and Mass derivatives).
