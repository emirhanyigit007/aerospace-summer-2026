# Mass-Spring-Damper System Simulation in Simulink

## Project Overview

This project will demonstrate the simulation of a basic second-order dynamic system using Simulink.

The modeled system is a mass-spring-damper system, which is commonly used to understand vibration, damping, and transient response in engineering systems.
